# Industrial Scheduling Optimization

Проект по оптимальному распределению инженерно-технических работ между подразделениями с учетом ресурсных, сезонных, календарных и стоимостных ограничений.

Проект основан на моей выпускной квалификационной работе по профилю «Математическое моделирование и анализ данных в экономике».

## Постановка задачи

При составлении годового плана необходимо распределить инженерно-технические работы между подразделениями и определить сроки их выполнения.

При этом требуется учитывать:

- специализацию и допустимость исполнителей;
- ограниченный фонд рабочего времени;
- сезонные ограничения;
- полевую и камеральную части работ;
- продолжительность командировок;
- возможность объединения нескольких работ в один выезд;
- производственный календарь;
- трудовые и командировочные затраты.

Цель проекта — получить допустимое календарное распределение всех работ и минимизировать совокупную стоимость их выполнения.

## Исходный масштаб задачи

В исходной производственной постановке рассматривались:

- 1 290 работ годового плана;
- 50 видов инженерно-технических работ;
- 5 диагностических подразделений;
- 24 направления командировок;
- ресурсные, сезонные, календарные и стоимостные ограничения.

Исходные корпоративные данные в публичном репозитории не размещаются.

## Этапы решения

### 1. Подготовка данных

Данные по работам, подразделениям, допустимым исполнителям, сезонности, трудоемкости, производственному календарю и командировочным расходам объединяются в единую расчетную структуру.

Для каждой работы определяются:

- допустимые подразделения;
- допустимые сезоны;
- полевые и камеральные часы;
- направление командировки;
- стоимость выполнения;
- необходимые ресурсы.

### 2. Формирование кандидатных полевых групп

Несколько совместимых работ могут быть объединены в одну командировку, если они:

- относятся к одному направлению;
- имеют общего допустимого исполнителя;
- могут выполняться в один сезон;
- не превышают ограничение по длительности командировки.

Для формирования наиболее перспективных сочетаний использовался эвристический поиск с последующим отбором групп по расчетной экономии.

В исходной постановке:

- сформировано 203 806 допустимых групп;
- после отбора в оптимизационную модель передано 40 002 кандидатные группы.

### 3. Сезонная MILP-модель

Базовая модель построена как задача смешанного целочисленного линейного программирования.

Модель определяет:

- выбранные группы полевых работ;
- подразделение-исполнителя;
- сезон выполнения;
- назначение камеральной части;
- расчетное количество командировок.

Целевая функция минимизирует сумму:

- стоимости полевых работ;
- стоимости камеральных работ;
- командировочных расходов.

Основные ограничения:

- каждая работа должна быть выполнена;
- полевую и камеральную части выполняет одно подразделение;
- назначение соответствует специализации исполнителя;
- соблюдаются сезонные ограничения;
- нагрузка не превышает фонд рабочего времени.

Для реализации использовались `PuLP` и решатель `HiGHS`.

### 4. Календарная детализация

Сезонно допустимое решение не всегда можно напрямую разместить по конкретным рабочим датам.

Поэтому после решения базовой модели выполняется календарная проверка:

- выбранные группы объединяются в командировки;
- командировки размещаются в производственном календаре;
- учитывается дневной фонд рабочего времени;
- камеральная часть назначается после завершения полевой;
- выявляются перегрузки и неразмещенные работы.

### 5. Улучшенная CP-SAT-модель

Для получения полностью реализуемого расписания построена календарная модель на `OR-Tools CP-SAT`.

В ней единицей выбора является кандидатная командировка с заранее определенными датами. Модель одновременно учитывает:

- календарные интервалы;
- дневную доступность сотрудников;
- численность подразделений;
- очередность полевой и камеральной частей;
- продолжительность командировок;
- стоимость итогового решения.

## Результаты

В исходной производственной постановке модель позволила:

- распределить все 1 290 работ;
- получить расписание без превышения дневного фонда рабочего времени;
- сократить расчетное количество командировок с 264 до 149;
- снизить расчетную стоимость годового плана с 90,646 до 81,227 млн рублей;
- получить расчетную экономию 9,419 млн рублей, или 10,39%.

Результаты проекта могут использоваться для подготовки годового плана, сравнения сценариев распределения и обоснования управленческих решений.
